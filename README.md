# PGO_Kawiarnia
Niewielka kawiarnia działająca przy coworku przyjmuje zamówienia od stałych klientów. Właściciel chce mieć prostą aplikację konsolową, która pozwoli rejestrować produkty z menu, dane klientów i pojedyncze zamówienia. System ma być mały, czytelny i wygodny do rozwijania krok po kroku.
